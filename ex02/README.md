# 第2回
## 超高機能電卓（ex02/calc.py）
### 追加機能
- 四則演算：＋以外の四則演算を行う
- クリア:entryに入力されている数字・文字の一番右を消去する。(一番新しく入力された文字が消える)
- オールクリア：entryに入力されている数字・文字全てを消去する
- π:円周率を数値で表示させる
- √:ルートの計算をする
- エラー:記号のみまたは連続している場合、もしくは0で割っている場合は、Errorと表示される。

### 追加(機能以外)
- 数字とそれ以外のボタンの色を分けた。(数字：whitesmoke #f5f5f5 その他：silver #c0c0c0)
